About this project

This project represents a beginner-friendly frontend web application designed to simulate a real-world authentication system. It consists of a landing page that introduces the platform, along with dedicated login and signup pages for user access.


The focus of this project is on creating a visually balanced layout, maintaining clean design principles, and ensuring smooth user interaction. It serves as a foundational step toward building more advanced full-stack applications in the future.
