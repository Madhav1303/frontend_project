# Frontend Application Project

## 📌 Project Overview
This is a **basic frontend project** built using **HTML, CSS, and JavaScript**.  
The project consists of a **Landing Page**, **Signup Page**, and **Login Page**.

The main goal of this project is to demonstrate:
- Form handling and validation using JavaScript
- Basic navigation between multiple HTML pages

This project is **purely frontend-based**.  
No backend or database is used, and user data is **not stored permanently**.

---

## 🧩 Pages Included
- **landingpage.html** – Main entry page with navigation buttons  
- **signup.html** – User registration form with validation  
- **login.html** – User login form with validation  

---

## 🚀 Features

### 🔹 Landing Page
- Displays project information
- Contains **Signup** and **Login** buttons
- Buttons redirect to their respective pages

---

### 🔹 Signup & Login Pages
- User input fields:
  - Name
  - Email
  - Password
- Input validation using Regex expression in **JavaScript**
- Appropriate message is display after successful validation

---

## ✅ Form Validation Rules (JavaScript)

The following validations are implemented:

- **Email**
  - Must contain `@`
  - Must contain a valid domain (example: `.com`)

- **Password**
  - Must be at least **8 characters long**
  - Must contain atleast <b>one Capital letter</b>
  - Must contain atleast <b>one small letter</b>
  - Must contain atleast <b>one special character</b>

If validation fails, the user is alerted with appropriate messages.

---
